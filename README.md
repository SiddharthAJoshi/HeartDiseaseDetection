# HeartDiseaseDetection
•	Designed GUI which contained button and text boxes for data analysis and prediction of Heart disease using Tkinter package in Python.

•	Developed Python code for training and testing data using Pandas library for Gaussian Naves Bayes Classifier and Logistic Regression.

•	Implemented code for plotting the confusion matrix for calculating accuracy of the trained model and used Matlib library for plotting learning curves for graphical representation of the accuracy achieved.
